<h1 style="text-align: center;">Sagan Sync</h1>
Sagan Sync is a CLI tool that lets developers deploy code from their local machine to a Linux VPS with automatic HTTPs, file syncronization, and service restarts. Perfect for staging enviroments, bots, APIs, backend apps or fast MVPs.
